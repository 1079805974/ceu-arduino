Modes
=====

Polling Mode
------------

The *polling mode* of Céu-Arduino .

- 100% cpu

### Compilation

Examples that use interrupts have to be compiled with `CEU_ISR=true`:

```
make CEU_ISR=true CEU_SRC=samples/isr-01.ceu
```

Interrupt Mode
--------------

The *interrupt mode* of Céu-Arduino .

- [a,b]
- scope
